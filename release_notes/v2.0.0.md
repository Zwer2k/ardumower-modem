# ArduMower Modem v2.0.0

This release focuses on improving telemetry, control, and release infrastructure.

## Highlights

- Added support for ESP32-S3 builds with a separate `esp32` and `esp32-s3` release flow.
- LiveMap now shows the mower position on an OpenStreetMap background.
- GOTO and joystick control are available on the mower map, not on the LiveMap.
- Browser gamepad support for remote driving, including PS4 and other controllers.
- New GPS dashboard with better position, satellite, and track handling.
- Prometheus metrics now work reliably and no longer crash the modem on boot.
- Improved MQTT/Home Assistant/ioBroker integration.
- Better OTA and STM32 firmware update support.
- Updated web UI compatibility for newer Svelte and Node.js versions.
- CI/release pipeline now produces clearly named assets and a proper release body.

## Notes

- The release includes both ESP32 and ESP32-S3 firmware assets.
- The web UI and firmware package now support improved socket, TCP, and GPS stability.
- Release notes are supplied in English for the tag release.

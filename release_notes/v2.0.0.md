# ArduMower Modem v2.0.0

This release improves firmware builds, telemetry, remote control, and release automation.

## Highlights

- Added support for both ESP32 and ESP32-S3 builds with separate release assets.
- LiveMap now displays the mower position on an OpenStreetMap background only.
- GOTO and joystick control are provided on the mower map, not on the LiveMap.
- Browser gamepad support for remote driving, including PS4 and other controllers.
- New GPS dashboard with better position, satellite, and track handling.
- Prometheus metrics now work reliably and no longer crash the modem on boot.
- Improved MQTT support including better Home Assistant and ioBroker integration.
- Better OTA and STM32 firmware update support.
- Updated web UI compatibility for newer Svelte and Node.js versions.
- CI/release pipeline now produces clearly named assets and uses a proper English release body.

## Notes

- The release includes both ESP32 and ESP32-S3 firmware assets and separate build artifacts.
- The web UI and firmware package now support improved socket, TCP, and GPS stability.
- Release notes are supplied in English for the tag release.

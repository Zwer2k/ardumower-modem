# Release Notes

## Switch to ESP32-S3
* Set ESP32-S3 as the main supported platform (standard ESP32 has too little memory)
* Separated build and release paths for esp32 and esp32-s3

## Joystick and Gamepad Control
* Added joystick control to the web UI
* Added browser-based game controller support (ESP32-S3 only supports BLE, not classic Bluetooth)

## Mower Map
* Add move area lines and rover position
* Add goto to position

## GPS Dashboard & Live Map
* GPS dashboard with Skymap and detail GPS informaitions
* LiveMap now only displays the mower on the world map

## OTA / STM32 / Firmware
* Improved OTA update paths
* Added flashing support for STM32-based rover
* Added reboot option for modem, rover, and GPS

## Web UI & Svelte
* Svelte update and UI style improvements
* Node.js 22 / Rollup plugin compatibility
* Improved WebSocket proxy and UI stability

## Prometheus & Monitoring
* Fixed boot crash related to Prometheus metrics
* Better heap/system metrics and more stable exporting

## MQTT / Home Assistant / ioBroker
* Expanded MQTT support
* More stable socket and TCP connections
* Better integration potential for Home Assistant / ioBroker

## CI / Release
* GitHub Actions pipeline for tag releases and nightly pre-releases
* Release assets are now clearly named and packaged
* Prepared release notes generation from Git log

## Bugfixes
* Stability improvements
* Numerous fixes for BLE encryption, socket communication, HTTP adapters, compilation warnings, firmware upload, terminal handling, and more

# Release Notes

## Goto Navigation (Firmware-side, Sunray patch required)
* Replaced unreliable client-side PD controller with firmware-side autonomous navigation
* New `AT+R,<x>,<y>` command in Sunray firmware — sets a free point and drives to it using Stanley controller + obstacle avoidance
* Mower motor state is preserved during goto (OFF before → OFF during/after, ON before → ON during/after)
* `navigateTo(x, y)` available via WebSocket API

## Documentation
* Added ESP32-S3 pinout documentation with SVG diagram (`docs/pinout.md`)
* Added custom pinout diagram showing GPIO assignments for modem operation

# Release Notes

## GPS Restart Encryption Fix
* Fixed `BLE:COMM CRC ERROR` when restarting GPS from the web UI (`/api/mower/rebootGps`). The HTTP adapter now delegates to `MowerAdapter::rebootGPS()`, which properly encrypts the command before sending — matching the encryption expected by the STM32 when `ENABLE_PASS` is active. Previously, the command was sent unencrypted, causing the STM32's decryption step to garble it and fail the CRC check.

## Mower Heading Fix
* Fixed heading direction arrow on the modem map (`MowerPosition.svelte`). The STM32 sends `delta` in radians (-π..+π); the SVG arrow function expects degrees (0=N, CW). Added the conversion `headingDeg = 90 - delta * 180 / π`.

## STM32 Firmware Update Stability
* Added idle callbacks during STM32 OTA flashing to keep the router loop running, preventing serial buffer overflow and communication stalls.
* All module loops (BLE, HTTP, UI, socket, MQTT, PS4 controller) are now paused during flashing via `MowerUpdater::isFlashing()` guard, eliminating conflicts with the firmware update process.

## Settings JSON Overflow Fix
* Increased `DynamicJsonDocument` pool size from 1024 to 2048 bytes in settings read/write and API handlers.
* Added overflow detection with error logging to catch future capacity issues.

## Settings Persistence Fix
* Fixed in-memory settings not being updated before saving (`ui_adapter.cpp`).
* Added software restart fallback via `ESP.restart()` when `ticker.h` is unavailable.

## WebSocket Improvements
* New WebSocket clients now receive a fresh stats request immediately upon connection, ensuring the UI has up-to-date data without waiting for the next poll interval.

## StateCard Satellite Order
* Swapped display order of `visible_satellites_dgps` and `visible_satellites` in the status dashboard to match the Cassandra app convention (smaller DGPS count shown first).

## Router Fixes
* Fixed `expectResponseSince` initialization in `sendWithoutResponse()` to properly track command timing.
* Reduced verbose hex logging on the serial TX path.
